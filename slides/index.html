    3. Lowercase everything
    4. Remove stopwords
    5. Aggregate words by stem

Term Document Matrix

  • Exactly like a database table
  • Each DOCUMENT in the CORPUS is a row
  • Each TERM in the entire CORPUS is a column
  • Each cell is the frequency of the TERM in the DOCUMENT
